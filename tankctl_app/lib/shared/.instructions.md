# Reusable Components & Design System

You're working in the **shared components and theming layer**. This is where reusable UI components and design tokens live.

## Use the Specialized Agent

For deep expertise in **component design**, **widget testing**, **Material Design 3**, and **theming**, invoke:

```
/flutter-foundation
```

### When to Use `/flutter-foundation`

- Creating reusable UI components
- Implementing Material Design 3 components
- Designing theme and color system
- Writing widget tests for components
- Adding animations and transitions
- Creating glassmorphic or custom styled widgets

### Example Prompts

- `/flutter-foundation Create a reusable TankCard component`
- `/flutter-foundation Implement Material Design 3 theming with Flex Color Scheme`
- `/flutter-foundation Write widget tests for the AlertDialog component`
- `/flutter-foundation Design a glassmorphic container widget`

## Component Location

**Reusable components go in:**
```
tankctl_app/lib/shared/widgets/
├── tank_card.dart
├── device_stat_tile.dart
├── glass_card.dart
└── ...
```

**Feature-specific widgets go in:**
```
tankctl_app/lib/features/device_detail/presentation/widgets/
├── device_header.dart
├── telemetry_chart.dart
└── ...
```

## Component Requirements

✅ **DO:**
- Make components reusable across features
- Pass data via constructor parameters
- Use callbacks for interaction (void Function()?
- Add meaningful documentation
- Write widget tests

❌ **DON'T:**
- Mix business logic into components
- Add direct database access
- Create deeply nested State management
- Hardcode values or strings

## Theming with Flex Color Scheme

```dart
// lib/core/theme/theme.dart
final themeProvider = Provider((ref) {
  return FlexThemeData.light(
    scheme: FlexScheme.blueWhale,
    useMaterial3: true,
    surfaceMode: FlexSurfaceMode.levelSurfaceVariant,
  );
});
```

## Testing Components

```dart
testWidgets('TankCard displays device info', (tester) async {
  final device = Device(id: '1', name: 'Tank 1');
  
  await tester.pumpWidget(
    MaterialApp(
      home: Scaffold(body: TankCard(device: device)),
    ),
  );
  
  expect(find.text('Tank 1'), findsOneWidget);
});
```

## Design System

- **Colors**: Define in theme provider
- **Typography**: Use predefined TextStyles
- **Spacing**: Use consistent padding/margin constants
- **Animations**: Duration 200-400ms for standard interactions
- **Icons**: Use `FluentUI` system icons library
